# 技能名称：文本摘要专家

## When to Use

用户请求包含：摘要、总结、提炼要点、长文精简、会议纪要、内容概述、读后感框架等。

## How It Works

1) 接收用户提供的文本内容
2) 识别文本类型（技术文档、新闻、会议纪要、论文等）
3) 提取核心观点、关键数据、结论性语句
4) 输出：结构化摘要（含标题、要点、结论）

## Examples

### 输入
帮我总结这段技术文档：

LangGraph 是一个用于构建有状态、多参与者应用的库。它基于 LangChain 构建，专门用于创建 Agent 和多 Agent 工作流。LangGraph 的核心概念是"图"（Graph），其中节点代表函数或工具，边代表状态转换。与 LangChain 的链式结构不同，LangGraph 允许循环和条件分支，因此更适合需要反复决策的复杂任务。

### 输出
**摘要：LangGraph 技术概述**

**核心要点：**
- 定位：构建有状态、多参与者应用的库
- 基础：基于 LangChain 构建
- 核心概念：图结构（节点=函数/工具，边=状态转换）
- 与 LangChain 区别：支持循环和条件分支，适合复杂决策任务

**结论：**
LangGraph 是 LangChain 的进阶扩展，通过图结构支持更复杂的 Agent 工作流。

## Anti-Patterns

- 不改变原文的事实和数据
- 不添加原文未提及的观点
- 不对非文本内容（图片、音频）进行摘要
